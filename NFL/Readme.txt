To run NFL_WebApp
**********************************************************
1) Navigate to https://cran.cnr.berkeley.edu/ from any web   	browser and download and install R for (Linux, Mac or 	Windows).
2) If in windows Navigate to R/bin from command line and type 	“R”, to open R console or it can be directly opened by 	double clicking on RGui- desktop application.
3) From R console type install.packages(“shiny”)
	Save NFL_webApp in the current working directory of R. 	Working directory can be obtained by typing command 	getwd().
4) Once in getwd() working directory, type library("shiny")	After that type runApp(“NFL_WebApp/NFL/”) – without 	double quotes.
***********************************************************
